# BrewxellesAPI

A Node.js & Express API serving data about Brussels-based beers, styles and breweries.

## Project status
Scaffold only (no code yet). Initial setup with folders, .gitignore, and README.

## Requirements
- Node.js >= 20
- NPM

## Getting started
1. Clone the repo
2. Run 
pm install (once dependencies are added)
3. Create a .env based on .env.example

## Environment
Create .env using .env.example:
MONGODB_URI=
