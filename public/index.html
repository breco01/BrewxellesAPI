BrewxellesAPI

Node.js + Express + MongoDB

Welcome! Quick checks:

  • GET /api/hello – Hello JSON
  • GET /api/health – Healthcheck
  • Swagger UI (/docs) – interactive API documentation

Tip: Use query params like ?limit=5&offset=0, ?q=search, and sorting with ?sort=field&order=asc|desc.

Breweries

GET /api/breweries – list
  • Query: ?limit (default 10), ?offset (default 0), ?q (search in name/city), ?sort (name|city|createdAt), ?order (asc|desc)
  • Example: /api/breweries?limit=5&offset=0&q=brussels&sort=name&order=asc
GET /api/breweries/:id – details
POST /api/breweries – create
{
  "name": "Brasserie de la Mule",
  "city": "Schaerbeek",
  "website": "https://www.brasseriedelamule.be",
  "contactFirstName": "Joel"
}
PUT /api/breweries/:id – update
{ "city": "Brussels" }
DELETE /api/breweries/:id – delete

Beers

GET /api/beers – list
  • Query: ?limit, ?offset, ?q (search in name/description/style), ?style, ?brewery (by breweryId), ?sort (name|abv|createdAt), ?order (asc|desc), ?populate=true
  • Examples:
    /api/beers?limit=5&offset=0&q=ipa&sort=abv&order=desc&populate=true
    /api/beers?style=Gueuze&brewery=<BREWERY_ID>
GET /api/beers/:id – details (?populate=true supported)
POST /api/beers – create
{
  "name": "Gueuze 100% Lambic",
  "breweryId": "<BREWERY_ID>",
  "style": "Gueuze",
  "abv": 5.5,
  "ibu": 10,
  "description": "Traditional Brussels lambic blend."
}
PUT /api/beers/:id – update
{ "abv": 6.0 }
DELETE /api/beers/:id – delete

Validation (summary)

  • Breweries:
    • name, city: strings, length 2..100, required
    • website: HTTPS URL (e.g. https://example.com)
    • contactFirstName: optional, no digits, max 50 chars
  • Beers:
    • name: string 2..100, required
    • style: string 2..50, required
    • breweryId: must reference an existing Brewery (MongoDB ObjectId)
    • abv: number 0..20 (required), ibu: integer 0..120 (optional)
    • description: optional, max 500 chars
  • Query params: sort and order are validated; populate accepts true|false.

Quick curl examples

# Create brewery
curl -X POST http://localhost:3000/api/breweries \
  -H "Content-Type: application/json" \
  -d '{"name":"Cantillon","city":"Brussels","website":"https://www.cantillon.be"}'

# List beers (search + sort + populate)
curl "http://localhost:3000/api/beers?q=gueuze&sort=abv&order=desc&populate=true"

On Windows PowerShell, use Invoke-RestMethod or curl.exe and escape quotes if needed.
